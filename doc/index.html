PlaneShifter(planeContainer, camera, controls, mouseReference)

PlaneShifter is an helper for THREE.Object3D that contains 3 orthogonal planes. The purpose is to make an easy translation of the whole container along each plane's normal vector, and the same for rotation. The module THREE must be defined in advanced.
